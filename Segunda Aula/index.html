Lista com comandos do git

  • Git init - Cria um repositório git
  • Git add - Coloca os arquivos em stage
  • Git commit - Cria um commit com base com o que está no stage
  • Git push - Envia o commit para o repositório remoto
  • Git remote - Mostra a lista de conexões remotas do repositório
  • Git status - Mostra o que pode ser levado para a stage e o que está nela
  • Git log - Mostra o histórico de commits
  • Git branch - Cria uma branch
  • Git checkout - Troca para outra branch
