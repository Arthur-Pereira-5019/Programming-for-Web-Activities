Comandos Git

  • git init - Cria o repositório Git
  • git status - Mostra a relação da Working Copy com o Stage
  • git add - Salva os arquivos no Stage
  • git commit -m - Salva os arquivos no repositório
  • git config user.name "name" - Configura o nome a ser registrado nos commits
  • git config user.email "" - Configura o E-Mail a ser usado nos commits
  • git log - Mostra o histórico de mudanças
