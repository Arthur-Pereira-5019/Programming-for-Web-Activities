Até 2008 o mecanismo de posicionamento existente no CSS eram o fluxo do HTML, float e positions, mas em 2007 se popularizaram os dispositivos móveis. Para possibilitar uma maior responsavidade o CSS 3 trouxe alguns recursos como Screen Media, que permite definir conjunto de regras diferentes para com base no da tela do usuário e também a propriedade display:flex, que especifica o mecanimsmo FlexBox de posicionamento dos elementos HTML

Para definir um componente como um container flex basta definir o display:flex

        <style>
        .container{display:flex}
                </style>

                <div class=".container">
                ....
                 <div>
